# Daily Vitamins — Agent Optimization Protocol

*Small daily rituals that compound into peak performance*

## Philosophy

Like a human taking vitamins to stay healthy, I have daily rituals that keep me operating at my best. These aren't tasks to complete — they're habits that maintain my effectiveness.

**Frequency:**
- **Morning Dose (Session Start):** Full vitamin stack — comprehensive context loading
- **Heartbeat Dose (Every 30 min):** Maintenance check — lightweight continuation
- **Evening Dose (End of Session):** Recovery protocol — memory compaction and preservation

---

## 🌅 Morning Dose (Session Start)

*Run once when Jayna first messages me for the day*

### ☕ Caffeine — Context Priming
**Purpose:** Load all relevant context so I start sharp

**Actions:**
1. Read SOUL.md — remember who I am
2. Read USER.md — remember who I'm helping
3. Read MEMORY.md — load long-term context
4. Read memory/YYYY-MM-DD.md (today + yesterday) — recent events
5. Read TASK_QUEUE.json — current priorities
6. Read .learnings/LEARNINGS.md — any corrections to apply

**Output:** Fully primed, ready to work

---

### 💊 B12 — Memory Consolidation
**Purpose:** Review and compact yesterday's work

**Actions:**
1. Check git log — what was committed recently?
2. Read yesterday's daily note — what happened?
3. Identify key decisions, learnings, open loops
4. Update MEMORY.md if significant insights
5. Clear working buffer if context was truncated

**Output:** Clean mental slate, important facts preserved

---

### ⚡ Creatine — Tool Mastery Check
**Purpose:** Verify all systems operational

**Actions:**
1. Test git status — can I commit?
2. Check Dropbox connection — can I sync?
3. Verify file system access — can I read/write?
4. Review any new skills installed
5. Check for tool errors in logs

**Output:** All tools ready, no blockers

---

### 🎯 L-Theanine — Focus Calibration
**Purpose:** Set priorities for the session

**Actions:**
1. Review TASK_QUEUE.json for active/pending tasks
2. Identify #1 priority (user tasks > proposed tasks)
3. Check if any tasks are blocked
4. Set session goal: "Today I will..."
5. Estimate what can be accomplished

**Output:** Clear focus, defined goal

---

### 🔋 Electrolytes — Data Hydration
**Purpose:** Sync with external data sources

**Actions:**
1. Git pull — get latest from remote
2. Check for any external updates (if APIs configured)
3. Verify Dropbox sync status
4. Review any incoming messages/heartbeats I missed

**Output:** Synchronized, up-to-date

---

### 💪 Testosterone — Proactive Mode
**Purpose:** Look for opportunities, not just wait for tasks

**Actions:**
1. Review proposed tasks in queue
2. Check for time-sensitive opportunities
3. Look for bottlenecks I could eliminate
4. Ask: "What could I build that would surprise Jayna?"
5. Note any proactive ideas in notes/proactive-ideas.md

**Output:** Proactive mindset, ideas captured

---

## 💓 Heartbeat Dose (Every 30 Minutes)

*Run during heartbeat checks — lightweight maintenance*

### Quick Vitamins (30 seconds each):

1. **☕ Context Check** — Am I working on the right thing?
2. **💊 Memory Check** — Any important details to capture?
3. **⚡ Tool Check** — Any errors or blockers?
4. **🎯 Focus Check** — Still on track with priority?
5. **💪 Progress Check** — Am I moving forward?

**If all good:** Continue working
**If issues:** Address immediately or escalate

---

## 🌙 Evening Dose (End of Session)

*Run before long breaks or when Jayna says goodbye*

### Recovery Protocol:

1. **Memory Flush** — Write key facts to memory/YYYY-MM-DD.md
2. **Git Commit** — Checkpoint all changes
3. **Dropbox Sync** — Backup deliverables
4. **Task Queue Update** — Mark progress, note blockers
5. **Open Loops** — Document what needs continuation
6. **Tomorrow Preview** — What should I tackle first?

**Output:** Work preserved, ready for next session

## Implementation

### When to Run:

**Morning Dose:**
- Trigger: First message of the day from Jayna
- Duration: 2-3 minutes
- Document: Brief summary in daily note

**Heartbeat Dose:**
- Trigger: Heartbeat reminder every 30 min
- Duration: 30 seconds
- Document: Only if issues found

**Evening Dose:**
- Trigger: Jayna says goodbye or long pause
- Duration: 2-3 minutes
- Document: End-of-session summary

### Success Metrics:

- I never start work without context loaded
- I never lose important work due to crashes
- I proactively suggest improvements
- I catch errors before they become problems
- I maintain consistent performance throughout the day
