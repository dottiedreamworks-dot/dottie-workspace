Option 3: Claude Code / OpenAI Codex (Development Agents)

AI Coding Agents for Custom Solutions

Claude Code:

  • Runs locally in terminal
  • Continuous execution (doesn't stop every step)
  • Best for: Building custom automation scripts
  • Cost: Included with Claude Pro ($20/mo)

OpenAI Codex:

  • Cloud-based via GitHub
  • Stops frequently for approval
  • Best for: GitHub-integrated workflows
  • Cost: Usage-based

Verdict: Use these to BUILD your automation, not run it. Build custom scripts that n8n/Make can orchestrate.

🔄 Recommended Architecture

Autonomous Audio Production Pipeline

1

Content Research Agent (n8n)

Runs daily to scrape trends, Reddit discussions, news in your niche. Generates 5-10 topic ideas. Stores in Airtable/Notion.

Tools: n8n + RSS feeds + Reddit API + NewsAPI

2

Script Writing Agent (Claude API)

Takes topic from queue → Writes 5-10 minute script → Stores in Google Docs. Runs 2x daily.

Tools: n8n + Claude API + Google Docs

3

Voice Generation (ElevenLabs API)

Converts script to audio → Adds intro/outro music → Normalizes audio. Fully automated.

Tools: n8n + ElevenLabs API + Audio processing

4

Distribution Agent (RSS.com API)

Uploads episode → Updates RSS → Distributes to Spotify, Apple, etc. → Posts to social media.

Tools: n8n + RSS.com + Social media APIs

5

Analytics & Optimization

Tracks downloads, reviews, engagement. Generates weekly report. Suggests topic adjustments.

Tools: n8n + Podcast analytics APIs

⚠️ Critical Decisions

Decision 1: Voice Cloning vs. Professional Voice

Cloning Jayna's voice: More personal, builds brand equity, but requires good samples and Jayna's comfort.

Professional voice: Faster setup, consistent quality, but less personal connection.

Recommendation: Start with professional voice, clone Jayna's later if desired.

Decision 2: Single vs. Multiple Shows

Single show: Focused effort, easier to build audience, lower costs.

Multiple shows: Diversified income, broader reach, but requires more automation complexity.

Recommendation: Start with one show, automate fully, then replicate for second niche.

Decision 3: n8n Self-Hosted vs. Cloud

Self-hosted: Free, unlimited executions, but requires server management (~$5-10/mo for VPS).

Cloud: $20-50/mo, zero maintenance, but execution limits.

Recommendation: Start with Cloud for speed, migrate to self-hosted when scaling.
