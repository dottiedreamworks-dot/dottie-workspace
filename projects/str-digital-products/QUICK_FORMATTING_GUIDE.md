# QUICK FORMATTING GUIDE - STR Maintenance Pro System
## Apply This in 5 Minutes

---

## Sheet 1: Master Task List

### Step 1: Bold Headers
1. Click row number "1" on left side
2. Click **Bold** button (B) in toolbar
3. Select row 1 again
4. Click Fill Color (paint bucket) → Light Gray (#d9d9d9)

### Step 2: Freeze Header Row
1. Click **View** menu
2. Click **Freeze** → **1 row**

### Step 3: Conditional Formatting (Priority Colors)
1. Select column D (Priority)
2. Click **Format** → **Conditional formatting**
3. Add these rules:

**Rule 1 - High Priority:**
- Format cells if: **Text is exactly** → `High`
- Formatting: Light red background (#ffc7ce)
- Click **Done**

**Rule 2 - Medium Priority:**
- Click **Add another rule**
- Format cells if: **Text is exactly** → `Medium`
- Formatting: Light yellow background (#ffeb9c)
- Click **Done**

**Rule 3 - Low Priority:**
- Click **Add another rule**
- Format cells if: **Text is exactly** → `Low`
- Formatting: Light green background (#c6efce)
- Click **Done**

### Step 4: Auto-Resize Columns
1. Select all columns (click column A, hold Shift, click column K)
2. Double-click between any two column headers
3. All columns auto-fit to content

## Sheet 2: My Schedule

### Step 1: Bold Headers
1. Click row "1"
2. Click **Bold** button
3. Fill Color → Light Gray

### Step 2: Freeze Header Row
1. **View** → **Freeze** → **1 row**

### Step 3: Conditional Formatting (Status)
1. Select column E (Status)
2. **Format** → **Conditional formatting**

**Rule 1 - Done:**
- Format cells if: **Text is exactly** → `Done`
- Formatting: 
  - Background: Light green (#c6efce)
  - Text: Dark green
  - Check "Strikethrough"
- Click **Done**

**Rule 2 - Overdue:**
- Add another rule
- Format cells if: **Text is exactly** → `Overdue`
- Formatting:
  - Background: Light red (#ffc7ce)
  - Text: Dark red, Bold
- Click **Done**

**Rule 3 - In Progress:**
- Add another rule
- Format cells if: **Text is exactly** → `In Progress`
- Formatting:
  - Background: Light yellow (#ffeb9c)
- Click **Done**

**Rule 4 - Not Started:**
- Add another rule
- Format cells if: **Text is exactly** → `Not Started`
- Formatting:
  - Background: Light gray (#f2f2f2)
- Click **Done**

### Step 4: Date Formatting
1. Select column D (Due Date)
2. **Format** → **Number** → **Date**
3. Select column F (Date Completed)
4. **Format** → **Number** → **Date**

## Sheet 5: Dashboard

### Step 1: Title Formatting
1. Click cell A1
2. Type: `STR MAINTENANCE PRO DASHBOARD`
3. Make it **Bold**, size 14pt
4. Merge cells A1:E1 (select range, click Merge cells)
5. Center align
6. Fill Color: Dark blue (#4472c4)
7. Text Color: White

### Step 2: Section Headers
1. Select cells with section titles (A3, A7, A13, A19, A22)
2. Make them **Bold**
3. Fill Color: Light blue (#d9e1f2)

### Step 3: Number Formatting
1. Select column B (all numbers)
2. **Format** → **Number** → **Number**
3. This makes the counts display properly

### Step 4: Currency Formatting
1. Select cost cells in column B (rows 20, 23-34)
2. **Format** → **Number** → **Currency**

## 🎨 BONUS: Make It Pretty

### Add Borders
1. On each sheet, select all data
2. Click **Borders** button (grid icon)
3. Choose "All borders"

### Alternate Row Colors
1. Select all data rows (not header)
2. **Format** → **Alternating colors**
3. Choose a color scheme (light blue/gray recommended)
4. Click **Done**

### Column Widths
After auto-resizing, adjust these manually for best appearance:
- **Master Task List**: Column I (Notes) → Make wider
- **My Schedule**: Column B (Task Name) → Make wider
- **Vendor Directory**: Column E (Email) → Make wider
